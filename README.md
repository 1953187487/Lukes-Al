# Lukes-Al

> Autonomous Security Intelligence Platform

## v0.1 - First Light

This is the **v0.1 pre-release** of Lukes-Al, the first public build of the autonomous security intelligence platform.

### Features

- **AI Cortex** - Streaming AI assistant with configurable providers (OpenAI / DeepSeek / Custom API)
- **Strike Console** - Authorized security testing terminal with execution logging
- **Android FS Bridge** - Bind local folders for AI read/write access
- **Themes** - Quantum and Matrix visual themes
- **Data Hub** - Export/Import `.lukeal` data bundles

### Tech Stack

- Single-file HTML application (no build step required)
- Vanilla JS + CSS3 with glassmorphism UI
- Configurable AI backend via REST API

### Quick Start

1. Open `index.html` in a modern browser (Chrome 86+ recommended on Android)
2. Open **Core Settings** and configure your AI provider
3. Start using **AI Cortex** or **Strike Console**
